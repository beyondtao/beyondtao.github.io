rvm升级Ruby

August 1, 2016 in mac

第一步：安装rvm RVM:Ruby Version Manager,Ruby版本管理器，包括Ruby的版本管理和Gem库管理(gemset)

$ curl -L get.rvm.io | bash -s stable
$ source ~/.rvm/scripts/rvm

等待终端加载完毕,后输入：

$ rvm -v

如果能显示版本好则安装成功了。

rvm 1.27.0 (latest) by Wayne E. Seguin <wayneeseguin@gmail.com>, Michal Papis <mpapis@gmail.com> [https://rvm.io/]

列出ruby可安装的版本信息

$ rvm list known

ruby-]2.2[.4]
[ruby-]2.3[.0]
[ruby-]2.2-head
ruby-head

安装一个ruby版本

$ rvm install 2.3.0

列出ruby已安装的版本信息

$ rvm list

rvm rubies

=* ruby-2.3.0 [ x86_64 ]

# => - current
# =* - current && default
#  * - default

安装完成之后ruby -v查看是否安装成功。

$ ruby -v
ruby 2.3.0p0 (2015-12-25 revision 53290) [x86_64-darwin15]
